# Melgarejo Sandoval Oscar Santiago

## Objetivo del repositorio

Este repositorio tiene como finalidad almacenar y organizar las prácticas realizadas en la materia de Estructuras Discretas (Facultad de Ciencias, UNAM, 2026-I), utilizando herramientas como Git, GitHub y el lenguaje de programación Haskell.

Cada práctica se encuentra organizada en su respectiva carpeta para facilitar su consulta y evaluación.

## Versión de GHC

The Glorious Glasgow Haskell Compilation System, version 9.10.1
