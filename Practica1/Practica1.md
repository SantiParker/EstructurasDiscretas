## 1. ¿Cuáles son las principales diferencias entre Haskell y Racket?

Haskell es un lenguaje de programación puramente funcional, fuertemente tipado y con evaluación perezosa (lazy evaluation). Está diseñado para garantizar seguridad de tipos y alta abstracción matemática.

Racket, por otro lado, es un lenguaje derivado de Scheme (familia Lisp), multiparadigma y con tipado dinámico por defecto. Es ampliamente utilizado en el ámbito educativo y en investigación en lenguajes de programación.

Principales diferencias:

- Haskell tiene tipado estático fuerte; Racket es dinámico (aunque existe Typed Racket).
- Haskell usa evaluación perezosa; Racket usa evaluación estricta.
- Haskell está más orientado a aplicaciones formales e industriales; Racket es más académico y experimental.

## 2. ¿Cuáles son las principales diferencias entre Haskell y Java?

Haskell es un lenguaje funcional puro, mientras que Java es un lenguaje orientado a objetos.

Diferencias principales:

- Haskell evita efectos secundarios; Java los permite.
- Haskell usa evaluación perezosa; Java evaluación estricta.
- Haskell tiene inferencia de tipos automática avanzada; Java requiere declaraciones más explícitas.
- Java es ampliamente usado en la industria empresarial; Haskell tiene un uso más especializado.

## 3. ¿Por qué Haskell no ha alcanzado una adopción significativa en la industria?

Algunas razones incluyen:

- Curva de aprendizaje elevada.
- Paradigma funcional puro poco común en la industria.
- Ecosistema más pequeño comparado con Java o Python.
- Menor disponibilidad de desarrolladores capacitados.

Sin embargo, Haskell se utiliza en sectores específicos como fintech, compiladores y sistemas críticos.

## 4. Analogía entre Git y GitHub

Una analogía simple sería:

Git es como el motor de un automóvil que permite que el vehículo funcione (control de versiones local).  
GitHub es como el estacionamiento o la red de carreteras donde puedes compartir y coordinar el uso del automóvil con otras personas.

Es decir, Git es la herramienta y GitHub es la plataforma en línea que permite colaboración.
