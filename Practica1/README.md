# Práctica 01: Instalación de Haskell y uso de Git/GitHub

## 1. Objetivo de la práctica

Instalar correctamente el compilador de Haskell (GHC), familiarizarse con el uso básico de Git y GitHub, así como comprender la estructura inicial de un repositorio para el curso de Estructuras Discretas.

## 2. Tiempo requerido

El tiempo total para completar la práctica fue de aproximadamente 2 a 3 horas, incluyendo la instalación, configuración y resolución de problemas.

## 3. Comentarios y problemas durante la instalación

Durante la instalación de Haskell surgieron algunos inconvenientes relacionados con la configuración de variables de entorno y el reconocimiento del comando `ghc` en la terminal. Fue necesario verificar la instalación y reiniciar la terminal para que el sistema reconociera correctamente el compilador.

También fue necesario instalar Git y configurar el usuario mediante:

git config --global user.name  
git config --global user.email
